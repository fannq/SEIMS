00051 //Assumptions when using BIGTIFF - The BIGTIFF specification does not have these limitations, however this implementation does:
00052 // - The width and the height of the grid will be no more than 2^32 (4G) cells in either dimension
00053 // - No single TIFF strip or tile will contain more than 2^32 (4G) cells
00054 // - No single linear partition strip (process) will contain more than 2^32 (4G) cells
00055 // - No TIFF metadata tag may have more than 2^32 (4G) values
